# keyboard

逐字键盘 · 隐私政策与支持页面（静态站点）。

部署到 GitHub Pages 或使用自定义域名时，将 `privacy.html`、`support/` 的 HTTPS 地址分别填入 App Store Connect（隐私政策 URL、技术支持 URL）。

## 本地预览

用浏览器打开 `index.html`。

## 文件说明

- `index.html` — 首页  
- `privacy.html` — 隐私政策  
- `support/index.html` — 技术支持（路径为 `/support/`）  
- `assets/style.css` — 样式
